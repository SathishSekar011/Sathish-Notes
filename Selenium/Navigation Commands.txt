Navigation Commands:
=====================
	- Used to launch url and also to navigate from one url to another url
	- It is also used to navigate to previous url and forward url
	- We can also refresh the url

Syntax:
---------
	--> driver.navigate().to("url");
	--> driver.navigate().forward();
	--> driver.navigate().back();
	--> driver.navigate().refresh();

driver.get():
	--> open a webpage and wait untill the the page is completely loaded
	--> used when we open only a specific url

navigate.to();
	--> open a webpage but don't wait for page to load
	--> used when we want to do navigation like back , forward
